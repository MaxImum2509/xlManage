# Exemples xlManage

Ce répertoire contient des exemples et projets de démonstration pour xlManage.

## Projets disponibles

### vba_project/
Projet VBA complet pour tester l'import/export et l'exécution de macros.

Voir [vba_project/README.md](vba_project/README.md) pour plus de détails.

### tbAffaires/
Projet de démonstration pour le suivi et la gestion d'affaires via Excel.

Contient des modèles, des données et des extractions pour illustrer les capacités d'automation Excel de xlManage.

**Structure :**
- `data/` - Modèles et données de base (commentaires.xlsx, data.xlsx, modèle.xltx)
- `extractions/` - Exemples d'extractions de suivi d'affaires

## À venir

- `cli_examples/` - Exemples d'utilisation de la CLI
